WL-SSH
------

Author : Kuriens Maliekal
Date : 20th Oct 2015
Description : Script to SSH into running instances on various AWS accounts
Version : 1.0

Configuration:
--------------

# Script tested and working on Python 2.7.x
# Install boto3 python module using the command "pip install boto3"
# Use the AWS CLI to configure AWS accounts as per http://docs.aws.amazon.com/cli/latest/userguide/cli-chap-getting-started.html#cli-quick-configuration
# Ensure you have all the SSH keys under "~/keys" directory with 400 unix permissions.
# All good to go now.
