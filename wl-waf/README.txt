WL-WAF
------

Author : Kuriens Maliekal
Date : 4 March 2016
Description : Get list of client IP addresses that match a specific regexp on AWS WAF and block them based on count
Version : 1.0

Configuration:
--------------

Script tested and working on Python 2.7.x
Install boto3 python module using the command "pip install boto3"
Adjust the variables on the top of the script as per requirements
All good to go now.
